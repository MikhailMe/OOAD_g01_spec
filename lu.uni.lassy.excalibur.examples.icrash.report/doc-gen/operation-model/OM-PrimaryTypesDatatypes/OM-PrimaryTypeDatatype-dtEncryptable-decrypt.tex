\subsection{Operation Model for decrypt}

\label{OM-decrypt}


The \msrcode{decrypt} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{decrypt}{ used for decrypting the called object}

	\addrowheading{Parameters}
	\addnumbereddoublerow{}{AKey: dtKey}{string, which contains a secret key from a secret file} 

	\addrowheading{Return type}
	\addsinglerow{ptBoolean}

		

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{ after decryption of the object we get encrypted object}

	\end{operationmodel}
